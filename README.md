angular-base
============

Kick-off project for AngularJS with set project structure and basic modules (authentication, authorization, etc.)

## Tasks

#### Install dependencies

```shell
   $npm install
```

### Run unit tests

```shell
   $npm test
```

### Run e2e tests

```shell
   $npm start
   $npm run protractor
```

### Run application locally

```shell
   $npm start
```
